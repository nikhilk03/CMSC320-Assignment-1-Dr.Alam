Q1: Why might be consider using git?

Answer: Git is used for version control to track changes in code, collaborate with others, and manage different versions of a project. It helps keep a history of modifications, allows for branching and merging, and supports collaboration by integrating changes from multiple contributors.

Q3: What is the difference between merging and rebasing?

Answer: Merging preserves the complete history with a merge commit, while rebasing rewrites history to create a cleaner, linear timeline.
